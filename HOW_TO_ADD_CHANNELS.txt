КАК ДОБАВИТЬ НОВЫЙ КАНАЛ В МОНИТОРИНГ

ПОШАГОВАЯ ИНСТРУКЦИЯ

1. ОТКРОЙТЕ ФАЙЛ config.py
   - Найдите раздел SOURCE_CHANNELS
   - Это список каналов для мониторинга

2. ДОБАВЬТЕ НОВЫЙ КАНАЛ
   - Добавьте канал в список в формате "@username"
   - Пример: "@new_channel_name"
   - Поставьте запятую после предыдущего канала

3. ПРИМЕР ДОБАВЛЕНИЯ:

4. СОХРАНИТЕ ФАЙЛ
   - Нажмите Ctrl+S
   - Убедитесь что файл сохранен

5. ПЕРЕЗАПУСТИТЕ БОТА
   - Остановите текущий процесс (Ctrl+C)
   - Запустите заново: python main.py
   - Бот начнет мониторить новый канал

ВАЖНЫЕ МОМЕНТЫ:

- КАНАЛ ДОЛЖЕН БЫТЬ ПУБЛИЧНЫМ
- БОТ ДОЛЖЕН ИМЕТЬ ДОСТУП К КАНАЛУ
- ИСПОЛЬЗУЙТЕ ТОЧНЫЙ USERNAME (с @)
- НЕ ЗАБУДЬТЕ ЗАПЯТУЮ ПОСЛЕ ПРЕДЫДУЩЕГО КАНАЛА

ПРОВЕРКА РАБОТЫ:

1. Запустите бота
2. Посмотрите логи - должно появиться сообщение о доступе к каналу
3. Если канал недоступен - проверьте правильность написания
4. Если все ОК - бот начнет переписывать посты из нового канала

УДАЛЕНИЕ КАНАЛА:

Чтобы убрать канал из мониторинга:
1. Удалите строку с каналом из списка
2. Уберите запятую после предыдущего канала
3. Перезапустите бота
